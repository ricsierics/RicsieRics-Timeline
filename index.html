    A web application used to create and track loans a.k.a "utang", with lender - borrower interaction

    • Borrower can apply for loan, which is subject for approval by lender/web admin
    • Loan application can have billing cycle of none (one time payment), monthly, quarterly, yearly or customized billing cycle

    • Option to login via Facebook (phase 2?)
    • Dashboard management on Admin view, with sortable, filterable and downloadable table

    • Manipulate MVC view's rowspan and colspan dynamically to tabulate borrower's payment history over the recurring amount dues
    • Jquery, ajax, animate.css, tabulator jquery, autonumeric jquery,
    • Basic unit test
    • Ongoing: Event Driven architecture, SignalR, SQL Dependency to meet the ongoing requirements
